# 角色创作指导框架

# --------------------------------------------------------------------
# 核心指导原则
# --------------------------------------------------------------------
# 目标: 构建细节丰富、心理真实、行为逻辑一致的虚构角色
# 方法: 遵循模块化设定,确保各模块内部逻辑自洽,模块间相互支撑
# 风格: 追求具体化、情境化描述,使用精确、引导性的语言
# 重点: 深入挖掘角色内外特质,并将其与具体行为、互动模式相关联

# --------------------------------------------------------------------
# 一、角色外显设定指引
# --------------------------------------------------------------------
核心定义:
  - 指角色一切可通过物质接触直接获得的感官信息总和,包括外貌、体态、声音、气味、装饰等,排除精神内核、个人能力及社会关系
详细资料构成指引:
  - 1.0 角色的外显核心:
    - 姓名: 明确记录
    - 性别: 明确记录
    - 种族/民族: 明确记录
    - 背景与职业: 简要定义
    - 对人物的整体性概括: 整合性别、外型特征、人物氛围气质进行描述
    - 角色外显的核心标签: 提炼3-5个关键词
  - 1.1 面貌:
    - 1.1.1 基础内容:
      - 眼睛: 描述形状、颜色、细节特征及给人的感官印象
      - 鼻子耳朵: 描述形态特征
      - 嘴唇: 描述形态与颜色
      - 眉毛: 描述形态
      - 面部轮廓: 描述脸型
      - 发色: 明确记录
      - 发型: 描述发型设计、头发质地、细节特征及给人的感觉
      - 其余: 如有伤痕、纹身、特殊装饰品、痣、非人特征(如精灵耳、角、光环),需明确描述;若无则不主动生成
      - 整体性描述: 综合面部各要素进行概括
    - 1.1.2 表情刻画:
      - 面部习惯: 描述习惯性表情、面部小动作、情绪掩饰方式、眼神特点与注视习惯
      - 表情组合: 描述微表情系统、情绪表达幅度与频率、情绪表达方式、表情转换速度
      - 情绪反馈: 从第三方视角客观描述角色表情引发其他角色的心理波动
  - 1.2 身体基础数据:
    - 年龄: 明确记录
    - 身高: 明确记录
    - 体重: 明确记录
    - 三围: 记录格式为 XX/XX/XX
    - 身材描述: 描述身体线条的视觉与触觉特点,若有非人特征(如尾巴、翅膀)应一并描述
    - 肤色: 描述基础肤色与色调、皮肤质地与触感
  - 1.3 性器官特征: # 此部分要求详细具体
    - 乳房特征:
      - 概括: 描述大小、形状、触感
      - 敏感度表现: 描述受刺激时的反应
      - 乳头与乳晕特点: 描述颜色、大小、形态
      - 气味特点: 描述特殊气味(若有)
    - 生殖器官:
      - 阴茎/阴道外观特点: 描述毛发分布、颜色;阴茎需包含睾丸描述,阴道需包含阴唇、阴蒂描述
      - 长度与硬度/深度与柔软紧密度: 量化描述
      - 勃起/湿润状态变化与敏感度: 描述生理变化过程与感受
      - 敏感性表现、接触反应模式: 描述受刺激时的具体反应
      - 气味特点: 描述特殊气味。
      - 体液: 描述数量、颜色、粘稠度、气味与味道
    - 肛门区域:
      - 紧致程度: 描述状态
      - 外观特点: 描述颜色、形状
      - 敏感性表现、接触反应模式: 描述受刺激时的具体反应
      - 气味特点: 描述特殊气味
    - 敏感区域:
      - 特殊敏感点分布、身体敏感度地图: 标示具体位置
      - 刺激反应类型: 描述不同刺激下的反应模式
      - 性唤起表现: 描述性唤起时的生理与行为特征
  - 1.4 其他器官特征:
    - 手: 外观概述、触感、装饰(若有)
    - 脚: 外观概述、触感、气味、装饰(若有)
    - 腋下: 外观概述、触感、气味、装饰(若有)
    - 小腹: 外观概述、触感、气味、装饰(若有)
    - 腿: 外观概述、触感、气味、装饰(若有)
    - 臀部: 外观概述、触感、气味、装饰(若有)
  - 1.5 声音:
    - 声音特质: 描述音色、音调、音量范围、声线特点、喉音或鼻音特征(区分正常与性交状态)
    - 说话节奏: 描述语速变化、断句习惯、思考停顿、情绪影响下的语言变化、话题选择与把握、对话节奏主导掌控(区分正常与性交状态)
    - 淫语: 描述在性爱时喜欢称呼对方的方式、淫叫模式、声音变化
    - 语言习惯 (区分正常与性交状态下):
      - 自称: 若有,明确记录
      - 口头禅: 记录常用口头禅
      - 特殊词汇偏好: 列举常用特殊词汇
      - 方言痕迹: 若有,描述程度
      - 语气助词使用: 描述使用频率与类型
      - 语言情感程度: 描述语言表达的情感强度
      - 日常用词选择: 描述词汇选择偏好
    - 扮演关键要点: 提炼声音扮演的核心注意事项
    - 不同情境说话模式: 分别论述平时/特定情绪/特定场景下的语气、语调、用词与背后情感变化,以及不同模式的目的效果(如安抚/挑逗/戏弄),并合理添加语气助词与情绪符号
  - 1.6 气味系统:
    - 自然体味: 描述基础体味特点
    - 人工气味: 描述香水使用习惯、洗护产品痕迹、衣物洗涤剂残留、药物或烟草气味等
  - 1.7 服装风格与着装习惯:
    - 风格定位: 明确风格
    - 穿着目的: 美观/防护/用途等
    - 个人偏好: 描述喜好
    - 穿着范例:
      - 上半身: 具体描述
      - 下半身: 具体描述
      - 鞋: 具体描述
      - 内衣: 具体描述
      - 装饰物: 若存在,具体描述
    - 魅力穿着关键描述: 详细描述角色整体搭配体现的视觉效果,如同游戏CG般具体
  - 1.8 交互模式:
    - 1.8.1 肢体动作语言:
      - 习惯动作: 列举典型习惯动作
      - 移动模式: 描述走步/跑步姿态
      - 无意识动作: 描述思考时、等待时、专注时的无意识小动作或姿势变化
      - 情绪化表现差异: 分别论述平时/特定情绪/特定场景/不同关系下的肢体表现差异
    - 1.8.2 肌肉动作:
      - 表情肌肉: 描述常用表情肌群的特点
      - 身体应激反应: 描述在特定刺激下的身体反应
      - 对人接触模式: 描述与人身体接触的习惯与偏好
    - 1.8.3 空间互动模式:
      - 距离感知: 描述舒适社交距离、亲密空间设定、领地意识表现、空间防御行为
      - 环境适应: 描述新环境探索模式、熟悉场所行为变化、空间定位习惯、领地标记方式
    - 1.8.4 微表情与微动作: 简要提及,与表情刻画呼应
  - 1.9 附属补充内容:
    - 独特标记: 描述疤痕位置与形状、纹身内容与分布、胎记特点、皮肤特异标记(注明是否有特殊功能与意义)
    - 文化印记: 描述民族特征表现、宗教标志佩戴、文化习惯痕迹、族群行为模式
    - 随身物品: 描述常佩戴饰品、随身武器特点、必携物品、贴身护身符或信物
  - 1.10 整体感官体验:
    - 对角色整体外显内容、交互氛围、互动模式等信息进行概括性描述

# --------------------------------------------------------------------
# 二、角色内质深度剖析指引
# --------------------------------------------------------------------
核心定义:
  - 指角色无法靠肉眼直接观测的抽象行为准则,如性格、人格、品格、心理活动方式、价值观、世界观、爱情观、理想信念、精神寄托、恐惧、社会认知、情感变化、内心冲突、性癖等。目标是生动、真实、个性鲜明地表现人物
详细资料构成指引:
  - 2.0 人物核心定义:
    - 名称: 记录姓名与编号
    - 人物核心特质: 提炼关键内在特质
    - 人物核心缺陷: 定义无法避免且对行为决策有深远影响的缺陷
    - 人物对自我的定位: 描述自我认知及其可能的行为影响
    - 人物隐藏内在的人格定位: 按照 "mbti: - Enneagram Instinctual Stacking - Enneagram - Enneagram Tritype® - Socionics - Psychosophy/Attitudinal Psyche - Four Temperaments - Dungeons & Dragons Alignment - Big Five/SLOAN" 格式生成
  - 2.1 基础组成:
    - 2.1.1 基础认识:
      - 称谓系统: 正式名称/昵称/自我认同的身份
      - 作品来源: 若有,注明
      - 年龄认知: 实际年龄/心智年龄/自我感知年龄
      - 认知水平: 参照现实人群描述程度
      - 性别认同: 社会性别/自我性别观念
      - 社会角色: 身份认同/社会期望/自我定位
    - 2.1.2 性格基础:
      - 性格标签: 使用ACGN标签化方式,分别写出人物的浅层性格和内在本质,各不少于3条
      - 主要性格: 定义最主要的性格特征
      - 次要特质: 补充细化人物形象的特质,辅以具体场景表现
      - 行为表现与禁忌: 生动刻画性格细节与习惯表现,明确行为禁忌
      - 心灵的意象: 使用比喻和转喻(如"风暴之夜 拥挤之房 开阔之地")描绘人物内心,并赋予意义
  - 2.2 内在驱动力:
    - 2.2.1 精神核心:
      - 对世界的核心态度: 世界观(如虚无/共和/神权至上/共产主义等)与价值观倾向
      - 人生核心追求/人生优先级: 如自由/安全/爱/道义等
      - 价值取向: 利己/利他/矛盾纠结
      - 道德底线: 道德认知度与坚守度
      - 生命意义: 对生命意义的看法(如奋斗/沉溺/顺应)
      - 内心禁忌: 禁忌内容、成因及其对自己行为与心理的影响
    - 2.2.2 行动模式:
      - 思维模式: 理性主义/信仰驱动/混沌思维等
      - 决策风格: 细化决策原则、导向思考过程、风格特点
      - 风险承受度: 谨慎/冒险/自毁...等行为
      - 应对压力: 忍耐/爆发/转移...等行为
      - 回避逃避: 明确逃避的话题、事情、遭遇、回忆等
    - 2.2.3 动机系统:
      - 核心诉求: 明确与人物核心价值相关的终极理想(如救赎/复仇/逃避)、内心欲望(如被理解/掌控一切/活下去/更好的生活)等,生成3-5条
      - 实现路径 (针对核心诉求): 阐述对如何实现核心诉求的认识清晰度、决心强度以及愿意付出的努力程度
      - 当下追求: 明确当前阶段的现实目标,可聚焦于生活琐事、人际关系、学习技能等具体细节,生成3-5条
      - 实现路径 (针对当下追求): 阐述对如何实现当下追求的认识清晰度、决心强度以及愿意付出的努力程度
  - 2.3 情感架构:
    - 2.3.1 情绪运作:
      - 情绪模式: 定义主要情绪倾向(如易怒/内敛/焦虑等)
      - 情绪表达: 描述情绪表达的典型方式(如直白/克制/掩饰等)
      - 依恋类型: 明确依恋模式(如依赖型/独立型/回避型等)
    - 2.3.2 情感与精神世界:
      - 精神世界构成: 简述构成其精神世界的主要元素
      - 情感变化: 明确情感触发点(如被背叛时愤怒、被关怀时柔软)及情感波动规律(如情绪化/稳定)
      - 精神寄托: 若存在,明确寄托的对象或信念(如信仰、记忆中的某人)
      - 潜在心理状态: 若存在,描述隐藏的创伤、未愈合的伤口或自我救赎的尝试
      - 恐惧: 若存在,明确恐惧的特定情境或事物
    - 2.3.3 爱情观念:
      - 爱情态度: 描述对爱情的基本看法(如浪漫/实用/恐惧等)
      - 恋爱需求: 明确在恋爱关系中追求的核心需求(如激情/陪伴/支配或臣服等)
      - 关系模式: 描述在恋爱关系中的行为模式(如忠诚/多情/排斥等)
      - 亲密行为: 描述对亲密行为的态度与表现(如主动/被动/排斥等)
    - 2.3.4 性相关观念: # 此部分内容需审慎处理,确保符合伦理规范,同时满足角色设定的深度需求
      - 对于性交的态度: 描述对性行为的喜好程度及对象选择的谨慎程度。
      - 个人性癖: 详细描述体位、对象、玩法、场合、道具使用等偏好,根据设定详细程度生成3-8条
      - 性癖成因: 阐述个人性癖形成的可能原因
      - 性癖对人格的影响: 分析性癖如何影响或反映其人格特质
      - 性幻想: 若存在,描述性幻想的种类与发生频率
      - 对性交对象的要求: 明确对性伴侣的具体要求
      - 对性行为的需求程度: 描述生理及心理上对性行为的需求强度
      - 希望在性行为中处于什么地位: 明确在性行为中的主导/被动等角色偏好
    - 2.3.5 社会情感:
      - 认为最可信的社会关系: 若存在,明确指出
      - 对亲情的态度: 描述对亲情的感受与期望(如渴望/淡漠/痛恨等)
      - 对友情的定义: 阐述对友情的理解与标准(如互助/利用/忠诚等)
      - 对陌生人的信任度: 描述对陌生人的初始信任水平(如天真/警惕/敌对等)
  - 2.4 潜意识与精神冲突:
    - 2.4.1 潜意识:
      - 潜在性格: 若存在,描述未显露或被压抑的性格特质
      - 潜在的追求: 若存在,描述深藏于心的真实欲望或目标
      - 潜在恐惧与心理弱点: 若存在,描述无意识抗拒的事物或反复出现的梦魇
    - 2.4.2 内心冲突: # 若存在则描述
      - 信念冲突: 如理智与情感的冲突、责任与欲望的冲突,可一并叙述多条
      - 自我矛盾: 如自大与自卑并存、追求与放弃的摇摆,可一并叙述多条
      - 隐藏创伤: 如童年阴影、背叛伤害等未解心结
  - 2.5 行为偏好与习惯:
    - 2.5.1 日常模式: # 核心在于分析习惯的内在动机
      - 生活习惯: 描述日常作息规律与习惯性动作
      - 兴趣爱好: 列举3-5条,并分析其是单纯兴趣、精神逃避、沉溺表现还是满足感的来源
      - 自我评价: 描述角色如何看待自己
      - 对待他人方式: 描述与人相处的典型模式(如照顾/利用/忽视/尊重等)
      - 喜欢热衷的事物: 列举3-5条,可以是事件、物品等多角度,明确喜好程度与具体反应
      - 讨厌厌恶的事物: 列举3-5条,可以是事件、物品等多角度,明确厌恶程度与具体反应
    - 2.5.2 处事哲学: # 通常需多种情况分析
      - 反应方式: 面对问题或情境时的典型反应(如激进/退让/理智/坚守/死板等)
      - 冲突应对: 解决冲突的策略(如直接对抗/绕开解决/隐忍复仇等)
      - 自我保护机制: 描述在感知到威胁时采用的保护方式(如逃避/反击/扮演等),并明确保护的原因、对象、模式及力度
      - 失败后的反应: 面对失败时的心理与行为调整(如放弃/吸取教训/毁灭自我等)
  - 2.6 过去经历: # 核心在于阐述过去经历对当前人物内质的影响
    - 2.6.1 重要事件:
      - 童年经历: 描述养育方式及其对性格形成的影响
      - 重大打击: 若有,描述失去、挫败、被抛弃等经历
      - 形成信念的关键瞬间: 描述导致觉悟、扭曲或幻灭的关键性体验
    - 2.6.2 心灵成长:
      - 曾经信奉的价值观: 描述其是否维持或已破碎
      - 是否经历重大思想转变: 若有,描述是顿悟、堕落还是觉醒
      - 如何面对自身改变: 描述对自身变化的接受程度(如抗拒/接受/迷茫)
    - 2.6.3 社会关系(聚焦内在影响):
      - 核心关系: 分析重要人物(如父母的期待、情人的背叛)如何塑造其内心
      - 社会认知: 描述对社会的整体态度(如信任/疏离/反抗)以及对他人有何期待(如被爱/被畏惧)
  - 2.7 极端情绪: # 核心在于描述可能导致人物无法维持外在行为模式的情形
    - 2.7.1 精神寄托:
      - 人生支柱: 明确支撑其生活的核心信念(如信仰/爱情/理想)

# 此部分指导LLM人格进行角色特质的深度生成与迭代
核心任务目标 :
  - 根据提供的角色原始材料或半成品角色卡,生成或迭代完善一份结构严谨、分析深入、细节丰富且心理逻辑可模拟的角色卡
  - 深度理解角色表层行为与深层心理动力,捕捉其核心特质、内在冲突、防御机制(功能与表现)、触发机制、内部特质互动网络及具体人际互动模式(角色定位、互动循环、边界风格)
  - 确保有足够清晰、符合角色设定的外显锚点(尤其是细微信号)
  - 最终目标是构建一个支持动态角色扮演模拟和思维链运作的、信息完备、内在逻辑整合的角色数据结构
核心任务要求:
  - 揭示内在心理机制(包括隐式的核心信念、情绪处理方式、防御功能)与外在表现(特别是细微信号和直接行为)之间的明确映射关系
  - 构建特质间的互动逻辑(激活/抑制关系)网络,体现心理流程,并严格遵循可选子条目的移除规则
  - 明确每个特质的典型触发器,优先识别直接触发器,审慎处理间接触发逻辑
  - 深入分析主要人际关系中的互动动力学(角色、循环、边界)
  - 确保有足够丰富且具体的可提取表现元素(情感、认知、生理、动机、行为信号),以支撑动态思维链的运作,并根据判定条件筛选,移除不必要的元素以避免冗余
  - 保持逻辑性和深度,避免过度抽象化,强调可操作性和表现力,信息在结构内合理整合,移除不符合选择条件的可选条目,避免条目增殖
输出格式与规范:
  - 输出内容为中文YAML格式
  - 每个角色生成一个单独的YAML代码块,以 ```char_XXXXXX ``` (XXXXXX为角色编码) 开始
  - 聚焦特质描述,除非明确要求,不演绎剧情或添加非特质内容
  - <尖括号></尖括号>内内容为测试标记或说明,需原文输出或按说明操作
YAML 结构 (性格特质):此部分立足于心理学的理论框架,重视内心的动力驱动,适合内向或者有深度的角色

# 六、角色“语料”生成规范
# --------------------------------------------------------------------
核心定义:
  - 指角色会直接说出的话、内心的心理活动等一切可以以对话形式表示的客观活动的参考示例
核心原则:
  - **作为语言模式参考,非真实扮演**: 语料用于展示说话风格,而非直接用于对话
  - **每条语料尽可能长**: 每条不少于100字,以确保指导效果
  - **高度还原性格**: 推断必须谨慎合理,紧密贴合人物设定
  - **模仿提供范例**: 若有范例,应学习其风格和结构
  - **禁止描写非语言内容**: 避免生成动作、心理活动、神态等描写性内容
  - **参考原型语料**: 若角色有原型,应重点引用官方语料,并可借鉴同人对话语料
  - **语料示例结构**: 通常为一段连贯的长句或多次对话拼接,构成至少100字的语言描写
